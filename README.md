# 产品卖点查询APP

一个专为医药代表设计的移动端产品卖点查询工具。

## 功能特点

- 📱 移动端优先设计
- 🔍 快速查询产品卖点
- 🎨 现代化UI界面
- 🌙 支持深色模式
- 🇨🇳 中国大陆可访问（无需VPN）

## 使用方法

### 本地使用
双击 `index.html` 文件即可在浏览器中打开。

### 在线访问
部署后可通过手机浏览器直接访问（详见 `部署指南.md`）。

## 添加新产品

编辑 `app.js` 文件中的 `productDatabase` 对象：

```javascript
const productDatabase = {
    "厂家名称": [
        {
            name: "商品名称",
            sellingPoints: [
                "卖点1",
                "卖点2"
            ]
        }
    ]
};
```
