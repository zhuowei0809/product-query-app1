## 📋 部署步骤

### 第1步：注册Gitee
1. 访问：https://gitee.com
2. 点击"注册"，用手机号或邮箱注册
3. 登录账号

### 第2步：创建仓库
1. 点击页面右上角的 "+" → "新建仓库"
2. 填写信息：
   - 仓库名称：`product-query-app`（可自定义）
   - 权限：选择"公开"
   - 其他保持默认
3. 点击"创建"

### 第3步：上传代码（两种方式任选其一）

#### 方式A：使用自动脚本（推荐）
1. 在Gitee仓库创建完成后，复制仓库地址
   - 格式：https://gitee.com/您的用户名/product-query-app.git
2. 双击运行：`部署到Gitee.bat`
3. 按提示粘贴仓库地址
4. 输入Gitee的用户名和密码
5. 等待上传完成

#### 方式B：手动上传
1. 在Gitee仓库页面，点击"上传文件"
2. 选择并上传：
   - index.html
   - styles.css
   - app.js
3. 填写提交信息（随便写），点击"提交"

### 第4步：启用Gitee Pages
1. 在仓库页面，点击顶部菜单的"服务" → "Gitee Pages"
2. 点击"启动"按钮
3. 选择要部署的分支（通常是master）
4. 点击"启动"

### 第5步：获得访问地址
等待几秒后，会显示您的网站地址：
```
https://您的用户名.gitee.io/product-query-app
```

🎉 完成！这个地址可以在中国任何地方直接访问，无需VPN！
