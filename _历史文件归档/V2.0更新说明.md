# 产品卖点查询APP V2.0 更新说明

## 🎉 新功能：品类识别 + Get笔记API集成

### 更新内容

#### 1. 品类自动识别
- 每个产品现在都关联了品类信息
- 查询产品时会自动显示品类标签
- 当前支持的品类：
  - 抗生素（可乐必妥）
  - 肝病辅助用药（阿拓莫兰）
  - 心血管中成药（速效救心丸）

#### 2. Get笔记知识库集成
- 自动从Get笔记获取品类洞察和市场信息
- 实时加载，支持缓存机制
- 智能显示加载状态和错误提示

#### 3. 全新UI组件
- 品类标签：直观显示产品所属品类
- 品类洞察卡片：展示Get笔记提供的市场洞察
- 加载动画：提升用户体验
- 错误提示：友好的错误处理

---

## 📋 使用流程

1. **选择厂家** → 选择营销厂家
2. **选择商品** → 选择具体商品
3. **点击查询** → 查看产品卖点
4. **品类识别** → 自动显示品类标签
5. **品类洞察** → 自动从Get笔记加载品类信息

4. 解析并显示品类信息
5. 缓存结果以优化性能

##  添加新产品

编辑 `app.js` 文件，在 `productDatabase` 中添加：

```javascript
{
    name: "商品名称",
    category: "品类名称",  // 新增字段
    sellingPoints: [
        "卖点1",
        "卖点2"
    ]
}
```

## ⚠️ 注意事项

### Get笔记API限制
- QPS限制：2次/秒
- 日调用上限：5000次
- 目前处于公测阶段

### 数据要求
每个新产品必须添加 `category` 字段，否则不会触发品类洞察查询。

### 性能优化
- 已实现品类洞察缓存
- 同一品类只调用一次API
- 异步加载，不阻塞主流程
