# 产品卖点查询APP - 部署指南

## 🎯 部署到中国可访问的平台

### 方案一：使用 Netlify Drop（最简单，推荐）

1. 访问 [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. 将整个文件夹拖拽到网页上
3. 等待上传完成，立即获得一个 `.netlify.app` 域名
4. 免费且在中国大陆可访问

**注意**：Netlify官网在中国可能需要VPN访问，但部署后的 `.netlify.app` 域名在中国大陆通常可以正常访问。

---

### 方案二：使用 Vercel（需要安装CLI）

#### 安装 Vercel CLI
```powershell
npm install -g vercel
```

#### 部署
```powershell
cd "c:\Users\89164\Desktop\CSO课程\产品卖点查询APP"
vercel --prod
```

按照提示登录并完成部署，会获得一个 `.vercel.app` 域名。

---

### 方案三：使用国内平台（最稳定）

#### 3.1 阿里云 OSS 静态网站托管

1. 登录 [阿里云控制台](https://oss.console.aliyun.com/)
2. 创建 OSS Bucket
3. 在"基础设置"中开启"静态网站托管"
4. 设置默认首页为 `index.html`
5. 上传所有文件（index.html, styles.css, app.js）
6. 获得访问域名（如：`http://your-bucket.oss-cn-beijing.aliyuncs.com`）

#### 3.2 腾讯云 COS 静态网站托管

1. 登录 [腾讯云控制台](https://console.cloud.tencent.com/cos)
2. 创建存储桶
3. 开启"静态网站"功能
4. 上传文件
5. 获得访问域名

## 📱 移动端测试指南

### 本地测试
1. 在电脑上打开 `index.html`
2. 按 F12 打开开发者工具
3. 点击设备工具栏图标（Ctrl+Shift+M）
4. 选择手机型号（如 iPhone 12 Pro）
5. 测试界面和功能

## 🚀 快速部署命令（推荐）

如果您有 Node.js 和 npm，可以直接使用以下命令：

```powershell
# 安装 Vercel CLI
npm install -g vercel

# 进入项目目录
cd "c:\Users\89164\Desktop\CSO课程\产品卖点查询APP"

# 部署
vercel --prod
```

或者使用 Netlify CLI：

```powershell
# 安装 Netlify CLI
npm install -g netlify-cli

# 进入项目目录
cd "c:\Users\89164\Desktop\CSO课程\产品卖点查询APP"

# 部署
netlify deploy --prod
```
